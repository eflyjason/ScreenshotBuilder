# ScreenshotBuilder
A Python-based tool that helps you generate screenshots for App Store &amp; Google Play Store!

Still developing...

Main file: `main.py`

```
main.py -i <inputFile> -o <outputFile> -t "<text>"

optional input:
text color: -c "#ffffff"
font size: -s 100
font file: -f "Arial.ttf"
background color: -b "#000000"
```
